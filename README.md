# 🛍️ ZORO - Premium Men's Clothing Store

ZORO is a modern and stylish men's clothing e-commerce website designed with a premium black and gold theme. This project showcases a clean UI, admin dashboard, and basic shopping functionality.

---

## 🚀 Features

* 🖤 Premium dark theme (black + gold)
* 🛒 Add to Cart system
* 💰 Dynamic cart total calculation
* 🔐 Admin panel (add/delete products)
* 📦 Product management using localStorage
* ⚡ Fast and responsive design

---

## 🖼️ Preview

A sleek and luxury fashion UI inspired by high-end brands.

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage (for data persistence)
